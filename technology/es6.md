## 函数参数的解构赋值

### 给 x，y 赋默认值
* move 的参数是一个对象，通过对对象解构，得到 x 和 y 的值，解构失败，则 x 和 y 为默认值。

	```
	function move({x = 0, y = 0} = {}) {
		return x + y;
	}
	```

* 给 move 的参数指定默认值 {x:0, y=0}，如果参数为空的时候，则参数为默认值，通过参数解构，得到 x 的值为 0， y 的值为 0。如果有值，则使用该值进行解构。

	```
	function move({x, y} = { x: 0, y:0}) {
		return x + y;
	}
	```
